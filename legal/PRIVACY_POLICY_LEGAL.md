# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ

## SendBuddy — P2P платформа доставки посылок

**Версия:** 1.0  
**Дата вступления в силу:** 16 декабря 2025 г.

---

## 1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей платформы SendBuddy (далее — «Платформа», «Сервис»).

1.2. Платформа обязуется защищать персональные данные пользователей в соответствии с Федеральным законом № 152-ФЗ «О персональных данных» и настоящей Политикой.

1.3. Используя Платформу, вы даете согласие на обработку ваших персональных данных в соответствии с настоящей Политикой.

---

## 2. ОПРЕДЕЛЕНИЯ

2.1. **Персональные данные** — любая информация, относящаяся к прямо или косвенно определенному или определяемому физическому лицу.

2.2. **Обработка персональных данных** — любое действие с персональными данными, включая сбор, запись, систематизацию, накопление, хранение, уточнение, извлечение, использование, передачу, обезличивание, блокирование, удаление, уничтожение.

2.3. **Оператор** — Платформа SendBuddy, определяющая цели и содержание обработки персональных данных.

---

## 3. СОБИРАЕМЫЕ ДАННЫЕ

3.1. Платформа собирает следующие типы персональных данных:

### 3.1.1. Контактные данные:
- Номер телефона (обязательно)
- Имя
- Аватар (опционально)

### 3.1.2. Данные для верификации:
- Информация из социальных сетей (Telegram, VK) при использовании социального входа
- ID в социальных сетях

### 3.1.3. Данные о заданиях:
- Информация о созданных заданиях
- Маршруты доставки
- Фотографии предметов
- Описания посылок

### 3.1.4. Данные о транзакциях:
- Информация о платежах
- История доставок
- Суммы вознаграждений

### 3.1.5. Технические данные:
- IP-адрес
- Тип браузера
- Данные об устройстве
- Cookies

### 3.1.6. Данные о взаимодействии:
- Сообщения в чате
- Отзывы и рейтинги
- История активности

---

## 4. ЦЕЛИ ОБРАБОТКИ ДАННЫХ

4.1. Платформа использует персональные данные для следующих целей:

- Предоставление услуг Платформы (создание заданий, поиск курьеров, обработка платежей)
- Верификация пользователей и обеспечение безопасности
- Связь с пользователями по вопросам использования Сервиса
- Улучшение качества сервиса и разработка новых функций
- Показ целевой рекламы (с вашего согласия)
- Соблюдение требований законодательства
- Предотвращение мошенничества
- Разрешение споров

---

## 5. ПРАВОВЫЕ ОСНОВАНИЯ ОБРАБОТКИ

5.1. Обработка персональных данных осуществляется на основании:

- Согласия субъекта персональных данных (при регистрации)
- Выполнения договора, стороной которого является субъект персональных данных
- Соблюдения требований законодательства
- Защиты законных интересов Платформы

---

## 6. СПОСОБЫ ОБРАБОТКИ ДАННЫХ

6.1. Обработка персональных данных осуществляется с использованием средств автоматизации и без использования таких средств.

6.2. Обработка включает следующие действия:
- Сбор
- Запись
- Систематизация
- Накопление
- Хранение
- Уточнение
- Извлечение
- Использование
- Передача
- Обезличивание
- Блокирование
- Удаление
- Уничтожение

---

## 7. ЗАЩИТА ДАННЫХ

7.1. Платформа применяет следующие меры для защиты персональных данных:

- Шифрование данных при передаче (HTTPS/TLS)
- Хранение паролей в зашифрованном виде
- Ограниченный доступ к персональным данным только для уполномоченных сотрудников
- Регулярное обновление систем безопасности
- Резервное копирование данных
- Мониторинг несанкционированного доступа

7.2. Платформа не гарантирует абсолютную защиту данных от несанкционированного доступа, но принимает все разумные меры для их защиты.

---

## 8. ПЕРЕДАЧА ДАННЫХ ТРЕТЬИМ ЛИЦАМ

8.1. Платформа не продает ваши персональные данные.

8.2. Платформа может передавать данные третьим лицам только в следующих случаях:

### 8.2.1. Платежные системы:
- **ЮKassa** — для обработки платежей
- **CloudPayments** — для обработки платежей
- Передаются: данные о транзакциях, суммы платежей

### 8.2.2. SMS-сервисы:
- **Twilio** — для отправки кодов верификации
- Передаются: номер телефона, код верификации

### 8.2.3. Хранилище файлов:
- **AWS S3** — для хранения фотографий
- **Yandex Cloud Object Storage** — для хранения фотографий
- **Timeweb Cloud S3** — для хранения фотографий
- Передаются: файлы (фотографии посылок)

### 8.2.4. По требованию закона:
- При запросах уполномоченных органов
- При судебных решениях
- При необходимости защиты прав и безопасности

### 8.2.5. С вашего согласия:
- В других случаях с вашего явного согласия

8.3. Все третьи лица обязаны соблюдать требования защиты персональных данных.

---

## 9. ПРАВА ПОЛЬЗОВАТЕЛЯ

9.1. Вы имеете право:

- **Получать информацию** о ваших персональных данных
- **Требовать исправления** неточных данных
- **Требовать удаления** ваших данных (при отсутствии правовых оснований для хранения)
- **Отозвать согласие** на обработку данных
- **Ограничить обработку** данных
- **Подать жалобу** в уполномоченный орган по защите персональных данных (Роскомнадзор)

9.2. Для реализации прав обращайтесь через форму обратной связи на Платформе.

## 10. ХРАНЕНИЕ ДАННЫХ

10.1. Платформа хранит персональные данные в течение срока, необходимого для целей их обработки, или в течение срока, установленного законодательством.

10.2. После истечения срока хранения данные удаляются или обезличиваются.

10.3. Данные о транзакциях хранятся в соответствии с требованиями налогового законодательства.

---

## 11. COOKIES И АНАЛОГИЧНЫЕ ТЕХНОЛОГИИ

11.1. Платформа использует cookies и аналогичные технологии для:

- Улучшения работы Платформы

- Показа релевантной рекламы

11.3. Отключение cookies может повлиять на функциональность Платформы.
